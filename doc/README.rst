Agilent 7890/5975C-GC/MSD
-------------------------

气相色谱仪类型: 7890A

质谱仪类型: 5973C 通常情况下,不需要设计参数.

开机以及关机

a. 仪器参数的设定;
b. 分析方法的编辑;
c. 谱库的检索;
d. 报告的打印.

故障排查: 首先, 有两个泵, 一个在地上, 一个在仪器里面, 在仪器里面要看一下比较烫.
比较严重的故障:1. 真空泵反油, 主机器扳烧毁.
