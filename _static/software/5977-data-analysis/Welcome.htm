Site Prep

Before installing the software, you must complete the following items. Print this page for reference if necessary.

  1. Install MassHunter GCMS Acquisition. This step may be omitted for a Data Analysis only installation.
  2. Within Agilent GCMS Configuration, configure your GC/MS systems.

Note: Desktop icons for MSD ChemStation will not appear until the instrument is configured or reconfigured in Agilent GCMS Configuration.
